# My Portfolio

A personal portfolio showcasing graphic design and web development work.


## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** for animations
- **GitHub Actions** for deployment

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
