# Jack Carney — Portfolio

Personal portfolio showcasing graphic design and web development work.

**Live:** Deployed via GitHub Pages

## Sections

- **Hero** — Animated intro with text-shaped glow effect
- **Design** — YMUNK XIV poster designs, YMUNE XV branding & print materials, merchandise, and video montage
- **Design for America** — Event flyers, workshop graphics, and branding for DFA at Yale
- **Work** — Web development projects (DFA Yale, Haven Free Clinic sites) displayed in browser mockup frames
- **About** — Bio, education, current roles, and technical skills
- **Contact** — Email, LinkedIn, and GitHub links

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** for animations
- **GitHub Actions** for deployment to GitHub Pages

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Pushes to `main` automatically deploy to GitHub Pages via the included GitHub Actions workflow.
